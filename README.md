# Harmony Academy - School Website Project

A complete school website project built with HTML, CSS, and JavaScript, featuring a professional design with working dropdown menus and responsive layout.

## Project Overview

This project demonstrates a full-featured school website with:
- Login page for student/staff access
- Professional home page with navigation menu
- Working dropdown menus in the navigation bar
- Responsive design for mobile and desktop
- Form validation
- Smooth scrolling functionality
- Modern UI/UX design

## Features

### Login Page

- Clean form layout with input fields

## How to Use

1. **Open Login Page**: Open `login.html` in a web browser
   - Enter any username and password
   - Use the "Remember me" checkbox if desired
   - Click Login to test form validation

2. **Open Home Page**: Open `home.html` in a web browser
   - Click menu items to navigate
   - Hover over dropdown items (About Us, Academics, Admissions, Student Portal)
   - Click the "Explore Programs" button
   - Click on Student Portal > Login to navigate to the login page

## Responsive Design

The website is fully responsive with breakpoints for:
- Desktop (1200px and above)
- Tablet (768px - 1199px)
- Mobile (below 768px)

The navigation menu adapts for smaller screens with improved dropdown handling.
